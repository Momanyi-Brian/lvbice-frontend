# Lake Victoria Basin International Conference & Exhibition 2026 Website

## Overview

This project is the official website for the **Lake Victoria Basin International Conference and Exhibition 2026**, themed:

> **Harnessing Innovation and Emerging Technologies for Sustainable Development in the Lake Victoria Basin**

The website will serve as the primary digital platform for:

- Conference information and updates
- Speaker and session highlights
- Research and innovation showcase
- Registration and participation
- Sponsorship and partnership engagement
- Policy dialogue and networking opportunities

The design will follow a modern, premium conference experience, enhanced with smooth animations and interactive UI.

---

## Tech Stack

This website will be built with the following technologies:

### Core Framework

- **Next.js 15+ (App Router)**
- **TypeScript**

### UI & Styling

- **Tailwind CSS**
- **shadcn/ui** component library

### Interactive Components

- **React Bits** for modern section blocks

### Animation

- **GSAP (GreenSock Animation Platform)**
- Scroll-triggered transitions and motion effects
